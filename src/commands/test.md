# Запустить все тесты и исправить ошибки

## Обзор
Выполнить полный набор тестов и систематически исправить все ошибки, обеспечивая качество кода и функциональность.

## Шаги
1. **Запуск набора тестов**
   - Выполнить все тесты в проекте
   - Зафиксировать вывод и определить ошибки
   - Проверить как unit, так и интеграционные тесты

2. **Анализ ошибок**
   - Классифицировать по типу: нестабильные, сломанные, новые ошибки
   - Расставить приоритеты исправлений на основе влияния
   - Проверить, связаны ли ошибки с недавними изменениями

3. **Систематическое исправление проблем**
   - Начать с наиболее критичных ошибок
   - Исправлять по одной проблеме за раз
   - Перезапускать тесты после каждого исправления
